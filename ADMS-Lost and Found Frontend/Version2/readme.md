Advanced Database-Driven Lost and Found Object Management System
Frontend Version 2

Description
This project is a frontend implementation of a Lost and Found Object Management System. Version-2 focuses on CRUD operations and interaction with backend APIs.

Features
- View reported items (Read)
- Add new lost/found items (Create)
- Edit existing items (Update)
- Delete items (Delete)
- Success notifications
- Client-side validation using HTML5
- Data sent via POST/PUT requests (simulated)

Technologies Used
- HTML
- CSS
- JavaScript

How to Run
1. Download or clone the repository
2. Open `index.html` in any modern browser
